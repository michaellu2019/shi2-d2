# SHI2-D2 - THE PLAN

## Goal
- Create a high-fidelity simulation for a biped robot.
- Develop robust locomotion controllers for a biped robot.
- Build a capable biped robot that can walk.

## To Dos:
- Actuator layout and preliminary motor selection.
- Simplified dynamic and kinematic model.
- Robot block CAD.
- Simulation environment and software framework.
- Robot kinematic control in simulation.
- Robot locomotion controller in simulation.
- Detailed robot design with supporting analysis.

## Actuator Layout
- Neck Roll: LX-16 A Servo or Hi-Wonder 60 kg-cm Servo
- Neck Yaw: LX-16 A Servo or Hi-Wonder 60 kg-cm Servo
- Distal Neck Pitch: LX-16 A Servo or Hi-Wonder 60 kg-cm Servo
- Proximal Neck Pitch: LX-16 A Servo or Hi-Wonder 60 kg-cm Servo
- Hip Pitch: RobStride 00
- Hip Roll: RobStride 00 
- Hip Yaw: EduLite 05
- Knee Pitch: RobStride 01 or SteadyWin GIM8108-8 or SteadyWin GIM8108-9 or SteadyWin GIM8115-9
- Ankle Pitch: SteadyWin GIM3505-8
- Ankle Roll: SteadyWin GIM3505-8
